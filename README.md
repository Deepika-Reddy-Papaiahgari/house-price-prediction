# House Price Prediction using Machine Learning

## Overview
This project predicts house prices using Machine Learning techniques. A Linear Regression model is trained on the California Housing dataset to estimate housing prices based on features such as income, house age, rooms, population, and location.

---

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Flask

---

## Features
- Data preprocessing
- Model training
- House price prediction
- Model evaluation
- Deployment-ready Flask API

---

## Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation

---

## Model Used
- Linear Regression

---

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score
